# Data Dictionary

| Column Name | Description |
|---|---|
| CandidateID | Unique ID for each candidate |
| PositionID | Unique ID for each job position |
| JobTitle | Job role or position applied for |
| ApplicationDate | Date when the candidate applied |
| Source | Recruitment source through which the candidate applied |
| Stage | Current or recorded hiring stage of the candidate |
| DateatStage | Date when the candidate reached that stage |

## Column Notes
- **CandidateID** helps identify each applicant
- **PositionID** connects the candidate to a specific opening
- **JobTitle** shows the role being applied for
- **ApplicationDate** is useful for application trend and timeline analysis
- **Source** is used to measure hiring source effectiveness
- **Stage** is used for recruitment funnel and conversion analysis
- **DateatStage** helps calculate stage-wise time taken
